### 应用介绍：

---

#### * 搞笑字幕是一款“为搞笑的图片，创造有趣的字幕”的手机应用
             
#### * 广场：可以在广场中查看有趣的作品，可以保存到本地以及分享给好友

#### * 创作：可以在创作中选择有趣的素材或上传本地的素材进行创作

#### * 我的：可以查看我收藏的作品和我创作的作品
